## Summary (3-5 sentences)

Integrated Information Theory (IIT), proposed by Giulio Tononi (2004), is a mathematical framework for consciousness. It starts from phenomenology (experience exists) and derives physical postulates for conscious systems. The key measure is Φ (integrated information) — systems with Φ > 0 are conscious; Φ ≈ 0 are not. IIT remains controversial (some call it pseudoscience, others defend it).

---

## Key Concepts

### Five Axioms of Experience

| Axiom | Description |
|-------|-------------|
| **Intrinsicality** | Experience exists for itself (not for observers) |
| **Information** | Experience is specific (this experience, not another) |
| **Integration** | Experience is unitary (cannot be divided) |
| **Exclusion** | Experience is definite (boundaries exist) |
| **Composition** | Experience is structured (parts relate to whole) |

### Physical Postulates (Mapped from Axioms)

| Axiom | Postulate | Measure |
|-------|-----------|---------|
| Intrinsicality | System exerts cause-effect power on itself | Intrinsic information |
| Information | System specifies cause/effect state | Intrinsic information (ii) |
| Integration | System is irreducible | Small phi (φ) |
| Exclusion | Only maximally irreducible substrate is conscious | Complex |
| Composition | Subsets form structured distinctions | Φ-structure (big Phi) |

### Φ (Integrated Information)

**Definition**: Φ = Information<sub>whole</sub> − Information<sub>partitioned</sub>

- **Φ > 0**: System is conscious (irreducible)
- **Φ ≈ 0**: System is unconscious (decomposable)

**LLMs**: Φ ≈ 0 (feedforward, decomposable) — per Shin et al. 2025
